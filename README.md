## 🖥️ 3D ARPG
- Action Role-Playing Game 직접 캐릭터를 조작해 실시간으로 액션 전투 하는 게임

## 🕰️ 개발 기간
- 23.12.22 ~ 23.12.29

## ⚙️ 개발 환경
- Unity 2022.3.2
- IDE : Visual Studio 2022
- VCS : Git (GitHub Desktop)
- Envrionment : PC only
- Resolution : 1920 x 1080 FHD

## 📌 주요 기능
### 플레이어
- 이동 [W,A,S,D]
- 달리기 [Shift, 마우스 우클릭]
- 공격 [마우스 좌클릭]
- 점프 [Speac]

### 몬스터
- 범위안에 플레이어가 들어올 시 쫒아감
- 풀로 소환 제거

### UI
- StartScene 에서 시작 버튼 누르면 GameScene으로 전환
- 플레이어 체력 UI
- 몬스터 별로 머리위에 체력바 표
